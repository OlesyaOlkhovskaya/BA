Диаграмма последовательности: Регистрация пользователя @startuml actor Пользователь participant "Веб-приложение" as ВебПриложение participant "API" as API participant "База данных" as БД participant "Сервис отправки email" as EmailСервис Пользователь -> ВебПриложение: Открыть страницу регистрации ВебПриложение -> Пользователь: Отобразить форму регистрации Пользователь -> ВебПриложение: Отправить данные регистрации (имя, email, пароль) ВебПриложение -> API: POST /register (данные регистрации) API -> БД: INSERT INTO users (данные регистрации) БД --> API: Подтверждение успешного сохранения API -> EmailСервис: Отправить email подтверждения EmailСервис --> API: Подтверждение отправки email API --> ВебПриложение: Успешная регистрация ВебПриложение -> Пользователь: Уведомление о успешной регистрации @enduml
